Créer un hotspot Wi-Fi

Se connecter en wifi
sur le raspberry avec RaspAP

Sauvegarder le fichier de configuration du wifi

Faire un copie du fichier de configuration qui contrôle l'interface wifi :

sudo cp /etc/wpa_supplicant/wpa_supplicant.conf /etc/wpa_supplicant/wpa_supplicant.conf.sav
sudo cp /dev/null /etc/wpa_supplicant/wpa_supplicant.conf

Editer le fichier de configuration du wifi

Le fichier étant vide /etc/wpa_supplicant/wpa_supplicant.conf ajoutez les lignes suivantes :

ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1

Construire un point d’accès

Pour construire un point d’accès avec la raspberry pi, nous utilisons : le logiciel hostapd.
Et pour faciliter l’installation et l’administration de ce point d’accès on utilise RaspAP (une interface web qui nous permettra de contrôler notre hotspot) 

wget -q https://git.io/voEUQ -O /tmp/raspap && bash /tmp/raspap

Installer RaspAP dans le sous repertoire /var/www/html nommer le par exemple : RaspAP

Se connecter à votre hotspot Wi-Fi

Lorsque l’installation est terminée, la raspberry pi redémarre, et vous devriez voir apparaître un réseau « raspi-webgui » dans la liste des réseaux accessibles.
